# BxB 技能分类 JSON 数据结构文档

---

## characters_classified.json — 完整魔剑数据（含技能分类）

```json
[
  {
    "id":       int,      // 魔剣唯一ID
    "name":     string,   // 魔剣名称
    "rarity":   int,      // 稀有度: 3=★3, 4=★4
    "element":  int,      // 属性 (见下表)
    "type":     int,      // 武器种 (见下表)
    "states": {
      "改造" | "通常": {
        "skills": [
          {
            "name":      string,  // 技能名
            "effect":    string,  // 效果原文
            "bunrui":    int[],   // 效果分类，可多个 (见下表)
            "scope":     int,     // 效果对象范围: 0=自身 1=全体 2=限定全体
            "element":   int?,    // 仅当 scope=2 且限定属性时存在
            "type":      int?,    // 仅当 scope=2 且限定武器种时存在
            "condition": int      // 发动条件: 0=無条件 1=浑身 2=背水 3=破損
          }
        ],
        "bd_skill":   { "name": string, "effect": string, "cost": int },
        "basic_info": { ... }
      }
    }
  }
]
```

## scope 范围对照表

| scope | 含义 | element | type |
|-------|------|---------|------|
| 0 | 自身のみ | 无 | 无 |
| 1 | 味方全体（无限制） | 无 | 无 |
| 2 | 限定全体（属性或武器种） | 属性限定时存在 | 武器种限定时存在 |

## condition 发动条件对照表

| condition | 含义 | 触发文本特征 |
|-----------|------|-------------|
| 0 | 無条件 | — |
| 1 | 浑身（HP越多效果越高） | 残HPが多いほど / 損傷率が低いほど |
| 2 | 背水（HP越少效果越高） | 残HPが少ないほど / 損傷率が高いほど |
| 3 | 破損（HP低于50%触发） | 破損状態 |
